Crible+

Crible+ revisite le cribbage pour Android : parties rapides, mode hors ligne, règles claires. Un jeu de cartes pensé pour jouer partout, sans compte ni connexion.

Télécharger v1.6 (Android)

Installation : autorisez les sources inconnues dans les paramètres Android si demandé. APK signé — com.weidlerstudio.cribleplus.

Ce qui distingue Crible+

Hors ligne

Jouez sans connexion. Idéal en déplacement ou sans données mobiles.

Adversaire intégré

Affrontez l'ordinateur à votre rythme, avec plusieurs niveaux de difficulté.

Multilingue

Interface disponible en plusieurs langues pour un accès plus large.
